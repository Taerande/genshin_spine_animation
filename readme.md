#원신 Spine Animation

#msg event page 작성 룰

1. main 태그에 class를 msg\_\_wrapper로 지정한다.
2. main 내에 섹션은 section tag내 msg\_\_section을 지정한다.
3. nav 태그에 class를 msg\_\_navigator로 지정한다.
4. nav는 항상 ul > li > a href의 #로 id값을 지정한다.
5. 지정된 id값은 msg\_\_section의 id값과 일치시킨다.
